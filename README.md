The repository has been structured to ensure transparency and reproducibility of the research process.

- The **HTML version** provides a convenient way to inspect the complete project without installing any software.
- The **Jupyter Notebook version** enables full replication, verification, and extension of the analysis.

For evaluation purposes, it is recommended to consult the HTML version first, as it displays all code, outputs, visualizations, and results exactly as generated during the analysis.
